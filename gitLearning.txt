
SHOW ALL BRANCH : git branch

NORAML PUSH:
	1) git status
	2) git add .
	3) git commit -m "message"
	4) git push

Create New Branch Create Using Terminal:
	1) git status
	2) git branch new-feature ==> branch created
	3) git checkout new-feature ==> switch to branch
			(or)
		git checkout -b new-feature

Push New Branch in Terminal:
	1) git add .
	2) git commit -m "Your commit message here"
	3) git push origin new-feature

Git Pull On Master:
	1) git checkout master
	2) git pull origin master
